
Project Title: La Belle Couture – Online Clothing Store

Description:
La Belle Couture is a web application developed for a modern tailoring salon, offering clients the ability to browse clothing collections, view personalized designs, and book appointments directly through the platform. This project combines an elegant design with essential functionalities for managing orders and client interactions.

Technologies Used:
* Frontend: HTML, CSS, JavaScript
* Backend: C# with .Net
* Database: MySQL
